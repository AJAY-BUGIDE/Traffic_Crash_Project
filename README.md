# 🚦 Traffic Crash Analytics & Safety Intelligence Platform

## 📌 Project Overview

This project analyzes Chicago Traffic Crash data to identify accident patterns, high-risk locations, dangerous driving conditions, and contributing factors using SQL, Python, and Streamlit.

The project demonstrates advanced SQL querying, data analysis, Python-SQL integration, and interactive dashboard development.

---

## 🎯 Objectives

- Analyze traffic crash data using SQL
- Identify high-risk locations and conditions
- Generate business insights from structured data
- Implement advanced SQL concepts
- Build an interactive Streamlit dashboard
- Demonstrate real-world Data Analytics and Data Engineering skills

---

## 🛠️ Tech Stack

- Python 3.9+
- MySQL
- SQLAlchemy
- Pandas
- Streamlit
- VS Code

## 📈 SQL Analysis Performed

### Basic Analysis

- Total crash count
- Crash distribution
- Time-based analysis
- Common crash causes

### Advanced Analysis

- Top dangerous weather conditions
- Top injury-prone streets
- Injury percentage by crash type
- Peak crash hours by month
- Night-time crash analysis
- Daylight vs darkness comparison
- Traffic control device effectiveness
- Crash hotspots
- Injury rate analysis
- Year-over-year growth analysis
- Hotspot zone identification

### SQL Concepts Used

- GROUP BY
- ORDER BY
- Aggregations
- CASE Statements
- Window Functions
  - ROW_NUMBER()
  - RANK()
  - LAG()
  - NTILE()
- Common Table Expressions (CTEs)
- Subqueries
